В этом режиме Jenkins отображает ссылку на запуск JNLP на странице узла и виджете "Узлы сборки" в панели слева. Если включен соответствующий режим безопасности, ссылка будет доступна только авторизованным пользователям.
Запуск только со страницы узла. Вход не требуется
В этом режиме Jenkins отображает ссылку на запуск JNLP на странице узла. Ссылка будет доступна всегда, вне зависимости от того выключен ли узел.
ВНИММАНИЕ! В этом режиме защита выключена. Узел может быть перезапущен любым человеком имеющим доступ к Jenkins. Подчиненные узлы имеют привилегии для запуска произвольного кода на мастере. Не включайте этот режим, если вы не уверены в недоступности Jenkins для третьих лиц.
