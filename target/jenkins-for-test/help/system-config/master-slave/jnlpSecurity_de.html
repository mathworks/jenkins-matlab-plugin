Steuert das Starten von Slave-Knoten durch Jenkins über JNLP.
Starte nur von der Rechner-Detailansicht. Anmeldung notwendig, wenn Sicherheitsfunktionen aktiviert sind.
Dies ist die Vorgabe- und Normaleinstellung. In diesem Modus stellt Jenkins den JNLP-Startlink auf der Rechner-Detailansicht bereit. Sind Sicherheitsfunktionen aktiviert, ist der Link nur für authentifizierte Benutzer verfügbar.
Starte von der Übersichtsseite. Verlange Anmeldung, wenn Sicherheitsfunktionen aktiviert sind.
In diesem Modus stellt Jenkins den JNLP-Startlink sowohl auf der Rechner-Detailansicht als auch im Bereich "Build-Prozessor Status" bereit. Sind Sicherheitsfunktionen aktiviert, ist der Link nur für authentifizierte Benutzer verfügbar.
Starte nur von der Rechner-Detailansicht. Anmeldung notwendig, wenn Sicherheit aktiviert ist.
In diesem Modus stellt Jenkins den JNLP-Startlink auf der Rechner-Detailansicht bereit. Der Link ist verfügbar, wann immer der Slave-Knoten offline ist.
WARNUNG! In diesem Modus sind Sicherheitsfunktionen deaktiviert. Slave-Knoten können daher von jedem mit Zugriff auf den Server gestartet werden. Slave-Knoten haben die Möglichkeit, beliebigen Code auf dem Master auszuführen. Wählen Sie diese Option daher nur dann, wenn Sie sich der Risiken völlig bewußt sind!
